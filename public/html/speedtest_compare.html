Speed Test Analysis Report

Introduction

This report presents an analysis of internet speed test data before and after adding a new router. The goal is to evaluate the impact of the new router on download and upload speeds across different devices. The "before" results were collected every 60 seconds for one week to get a comprehensive aggregate of network behavior. The "after" results show significant improvements across all locations after just 24 hours, with download speeds doubling at a minimum.

First, we import the libraries and prepare the data into two seperate time periods

hosts = speedtest_data['hostname'].unique()

for host in hosts:
    data_before_host = data_before[data_before['hostname'] == host]
    data_after_host = data_after[data_after['hostname'] == host]
    
    if not data_before_host.empty and not data_after_host.empty:
        # Descriptive statistics
        print(f"Descriptive statistics for {host} before and after the new router:\n")
        print("Before:\n", data_before_host.describe(), "\n")
        print("After:\n", data_after_host.describe(), "\n")
        
        # Plot download speeds before and after
        plt.figure(figsize=(14, 7))
        plt.subplot(1, 2, 1)
        sns.boxplot(y='download_speed', data=data_before_host)
        plt.title(f'Download Speeds Before New Router for {host}')
        
        plt.subplot(1, 2, 2)
        sns.boxplot(y='download_speed', data=data_after_host)
        plt.title(f'Download Speeds After New Router for {host}')
        plt.show()
        
        # Plot upload speeds before and after
        plt.figure(figsize=(14, 7))
        plt.subplot(1, 2, 1)
        sns.boxplot(y='upload_speed', data=data_before_host)
        plt.title(f'Upload Speeds Before New Router for {host}')
        
        plt.subplot(1, 2, 2)
        sns.boxplot(y='upload_speed', data=data_after_host)
        plt.title(f'Upload Speeds After New Router for {host}')
        plt.show()
        
        # Perform t-test for download speeds
        t_stat_download, p_val_download = ttest_ind(data_before_host['download_speed'], data_after_host['download_speed'])
        
        # Perform t-test for upload speeds
        t_stat_upload, p_val_upload = ttest_ind(data_before_host['upload_speed'], data_after_host['upload_speed'])
        
        print(f"T-test for Download Speeds for {host}: t-statistic = {t_stat_download:.2f}, p-value = {p_val_download:.2e}")
        print(f"T-test for Upload Speeds for {host}: t-statistic = {t_stat_upload:.2f}, p-value = {p_val_upload:.2e}\n")
